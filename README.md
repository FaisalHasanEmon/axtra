## ⚙️ Installation

1. **Clone the Repository**

```bash
https://github.com/FaisalHasanEmon/axtra.git
cd axtra
```

2. **Install Dependencies**

Using npm:

```bash
npm install
```

3. **Start the Development Server**

```bash
npm run dev
```
